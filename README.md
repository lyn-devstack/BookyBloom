# BookyBloom 📚🌸

**BookyBloom** es una aplicación móvil nativa desarrollada en **Kotlin** diseñada para apasionados de la lectura. Es un espacio donde los usuarios no solo gestionan sus lecturas, sino que conectan con una comunidad para compartir reseñas, calificaciones y descubrir su próxima gran historia.

## ✨ Características Principales

* **Comunidad Literaria:** Interactúa con otros lectores a través de un sistema de comentarios y reseñas detalladas.
* **Perfiles Personalizados:** Cada usuario puede gestionar su identidad dentro de la app, incluyendo fotos de perfil y su historial de actividad.
* **Experiencia Visual Adaptable:** Soporte nativo para **Modo Oscuro**, ajustándose automáticamente a la configuración del sistema para una lectura cómoda.
* **Interfaz Intuitiva:** Diseño moderno siguiendo las guías de Material Design para Android.

## 🛠️ Stack Tecnológico

* **Lenguaje:** [Kotlin](https://kotlinlang.org/)
* **Arquitectura:** MVVM (Model-View-ViewModel) para un código limpio y mantenible.
* **UI:** XML / Material Components.
* **IDE:** Android Studio.
